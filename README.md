# DataBridge: Nonprofit Data Pipeline

An open-source data pipeline and analytics framework for nonprofit organizations.

## Overview

DataBridge helps nonprofits unify donor, volunteer, and campaign data from multiple sources into a centralized analytics platform.

## Tech Stack

- **Python**: ETL pipeline development
- **PostgreSQL**: Data warehouse
- **dbt**: Data transformations
- **Streamlit**: Analytics dashboard

## Project Status

Currently in Week 1 of development
