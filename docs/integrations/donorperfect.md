# DonorPerfect API Integration

## Overview
XML-based REST API for accessing donor and gift data

## Authentication
- Method: API Key
- How to get: Contact support@donorperfect.com
- Storage: .env file (DB_DONORPERFECT_API_KEY)

## Base URL
https://www.donorperfect.net/prod/xmlrequest.asp

## Key Endpoints
- dp_donorsearch - Search for donors
- dp_savegift - Create/update gifts
- dp_savedonor - Create/update donors
- Custom SQL - Run SELECT queries
